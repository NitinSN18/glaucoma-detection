# 🧠 Glaucoma Detection using Deep Learning

This project uses a deep learning model to classify fundus eye images into:
- Glaucoma
- Normal

## 🚀 Features
- EfficientNet-based CNN
- Transfer learning
- GPU support (MPS / CUDA)
- Data augmentation
- Model checkpoint saving

## 📊 Results
- Validation Accuracy: ~72%–99% (depending on training setup)
- Dataset: 564 training images, 141 validation images

## 🛠️ Tech Stack
- Python
- PyTorch
- torchvision
- EfficientNet
